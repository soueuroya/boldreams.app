# Boldreams | Software & Interactive Studio

Boldreams is an independent software and game development studio focused on creating mobile apps, games, VR/XR experiences, and scalable interactive software. We specialize in transforming complex technical requirements into polished, high-performance digital products.

## About The Studio

Led by **Daniel Boldrin**, a Senior Unity and Software Developer with over 14 years of production experience, Boldreams builds and ships:

- **Mobile Applications**: Native and cross-platform mobile apps built for high performance and seamless backend integration.
- **Game Development**: Full-cycle game development across mobile, PC, and console platforms.
- **Unity & XR Engineering**: Expert-level 2D/3D development, training simulations, and immersive spatial experiences optimizing memory, frame-rate, and scalability.
- **Multiplayer Systems**: Robust real-time networking, authoritative server architectures, and matchmaking designed for scale.

## Featured Projects

- **PetCity**: A virtual pet simulation focusing on rich interactions and long-term player engagement.
- **Hoame**: A lifestyle and meditation app designed with clean UX and robust backend state management.
- **Biom**: An interactive eco-system exploration title highlighting procedural generation.
- **Suprema Poker**: A massive-scale modular poker platform handling real-time matches and global union systems.
